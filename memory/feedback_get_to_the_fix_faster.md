Get to the fix faster. Don't ask the same question multiple times. Don't suggest things the user has already tried (cache clearing). When the symptom is clear, commit to a fix and offer to apply it.

**Why:** User was very frustrated by repeated questions and suggestions that didn't lead to a fix.

**How to apply:** When a symptom points to a likely cause, state it and offer the fix immediately. If wrong, adjust — don't loop through the same questions repeatedly.
